The compiled application can be run directly from the “bin” directory.
The “build” directory contains the project settings for visual c++ 2008 
express edition (freely available from http://www.microsoft.com/express/vc/).
The “lib” and “include” directories contain OpenCV libraries and headers, required for compilation.
The full version of the opencv library can be obtained from http://sourceforge.net/projects/opencvlibrary/.

Refer to include/EyeAPI.h for a list of possible functions.
The file main.cpp contains a usage example.
